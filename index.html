HTML5 Details / Summary Exploration

Basic A basic example of a <details> element in use. NOTES:
  • Not supported in IE or Edge (pre-Chromium)
  • Closed by default (use open marker attribute to override default behavior)
  • Keyboard accessible: tab between the sections; space or enter expands/collapses
  • When using <h1>, <h2> in <summary> elements, I used display: inline-block; to remove the awkward extra line-break
Open An exploration of the open attribute NOTES:
  • Demonstrates custom styles for open/closed content
  • Demonstrates hiding twisty arrow
  • Demostrates custom character for open/closed indicator
  • CSS is different for Firefox than Chrome, Safari, Opera
  • Hiding twisty arrow DOES NOT disable mouse/keyboard show/hide functionality
